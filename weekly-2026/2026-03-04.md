* 如期： LLVM 22.1.0 [was released](https://discourse.llvm.org/t/llvm-22-1-0-released/89950).

* 感兴趣的朋友可以看看： Anton Korobeynikov made a thread with [guidance for prospective GSoC 2026 applicants](https://discourse.llvm.org/t/gsoc-2026-projects-proposals-and-other-information/89967).

* 日益强化社区审核： Petr Hosek proposes [requiring pull requests for all commits](https://discourse.llvm.org/t/rfc-removing-the-require-pull-request-bypass/89993) (i.e. removing the currently available bypass). This does _not_ require review prior to commit for cases in which that is allowed by the developer policy.

* 日积跬步： In the RISC-V backend, code generation for the experimental P extension no longer requires an additional opt-in flag (beyond the flags needed to enable the extension).
  [118f64a](https://github.com/llvm/llvm-project/commit/118f64ad20d9).

* 爱好者狂喜： RISC-V FreeBSD kernel debugging support was added to LLDB.
  [4a602c0](https://github.com/llvm/llvm-project/commit/4a602c03ea05).

### 本周推荐阅读

本周推荐：《苏北人在上海》

作者：韩起澜（美国）

译者：卢明华

推荐：小编1号

中国基于地域性其实的各种尴尬笑话和歧视段子伴随着我们从小到大的日常生活，以至于被有意无意的忽略掉。这本书的作者作为一个外国人，尝试从各种文献报道和史料中进行挖掘，尝试还原一个特定的模糊的团体——“苏北人”——这个称谓是如何在过去百年中逐渐的形成亚文化群体。

忽略的时候不觉得有什么深刻，但是当学者开始认真的研究，历史的深度和悲悯心就无法压制的浮现了。清末到民国，自然条件恶劣，旱灾水涝频发，人民太苦了。
