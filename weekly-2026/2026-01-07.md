* AI辅助这个话题2026年估计也会火一整年： There's been continuing discussion on the newest LLVM AI tool policy thread, e.g. [PragmaTwice sharing practices from ASF communities](https://discourse.llvm.org/t/rfc-llvm-ai-tool-policy-human-in-the-loop/89159/13).

* 做后端的同学看看？ Bill Wendling proposed a solution for [improving LLVM support for alternative constraints in GNU extended inline assembly](https://discourse.llvm.org/t/rfc-improving-llvm-support-for-alternative-constraints-in-gnu-extended-inline-assembly/89350).  These are constraints like `"rm"` that allow the compiler to choose whether to use a general purpose register or memory for the operand, where currently the frontend defaults to a conservative choice. There's debae in the thread about whether introducing a new IR instruction is overkill and whether this can be handled in ISel.

* 传奇大哥稳定发挥： ChuanqiXu is [looking for contributors to help work through collected issues with C++20 modules on Windows](https://discourse.llvm.org/t/rfc-call-for-contributors-for-c-20-modules-on-windows/89342).

* 好耶，是社区熟悉的小伙伴： Yingwei Zheng [raised some questions about use of issue assignment](https://discourse.llvm.org/t/some-questions-about-the-issue-assignment/89334), especially (but not exclusively) in terms of how it relates to "good first issues".

* 祝贺： The first part of the LLVM ABI lowering library was committed - the ABI type system. [c2c787c](https://github.com/llvm/llvm-project/commit/c2c787c16f5d).

### 本周推荐阅读

本周推荐：《重新定义公司：谷歌是如何运营的》

作者：埃里克·施密特

译者：靳婷婷、陈序、何晔

推荐：小编1号

十年前的书籍，依然值得阅读。作者是编译器领域著名的词法解析工具 lex 的作者，也是 Google 在告诉发展阶段的 CEO，可以说跟两位创始人一起塑造了 Google 和 Alphabet 公司的辉煌。在目前跌中美AI大战中也是具有重要影响力（能决定千亿资金投向）的人物。

书中介绍的公司治理理念、方法、原则、会议技巧、决策流程注意事项，我看了之后都很有感触。2015年之前的谷歌真的是技术人员的向往之地。发现顶尖的人才、围绕顶尖人才创造合适的环境，并且尊重规律。我正在过去十几年的技术团队管理中逐渐积累的经验，与书中的观点遥相呼应，让我看到了很多相同的观点描述。

不过，有一点细节我要提醒各位读者，就是当我们深入去了解这些计算机历史上我们尊敬的领袖和技术导师、从他们的产品、代码、论述中汲取智慧的时候，可能会遗憾地看到，这种因为知识而产生的亲近感是单方面的。影响了硅谷和中国IT科技圈子的众多领袖，包括《从零到一》的作者、谷歌的创始人、美国AI头部企业的创始人，对于中国都公开表达过敌对 态度。当我第一次看到相关的英文视频访谈的时候，我的脑海中回想起了马斯克在SpaceX成功之前，被自己所崇拜的英雄、昔日的登月宇航员公开否定时候，脸上那种难以描述的忧伤。即便如此，我依然尊重每一位让我更加智慧的老师，并心怀感激。
