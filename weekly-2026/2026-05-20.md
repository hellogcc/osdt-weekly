### LLVM/Clang/LLDB/LLD

本节内容来自 [LLVM Weekly 第 646 期](http://llvmweekly.org/issue/646)，
[Alex Bradbury](https://www.linkedin.com/in/alex-bradbury/) 大哥持续稳定输出。

* 得看看： Nikita Popov made a proposal, '[yet another strict FP](https://discourse.llvm.org/t/rfc-yet-another-strict-fp/90798)'. "I believe that if we want to improve on the current strict FP situation, and treat strict FP as more of a first class citizen, we need to avoid further half measures, and actually support first-class FP environment annotations on all floating-point operations, including basic FP instructions." This proposal was move the IR from something like `%res = call float @llvm.experimental.constrained.fadd.f32(float %a, float %b, metadata !"round.dynamic", metadata !"fpexcept.strict")` to `%res = fadd float %a, %b fpenv(rounding_mode: dynamic, strict_except: true, ...)`.

* 注意看看： The Project Council fed back on some RFCs, such as [enforcing the rule of three](https://discourse.llvm.org/t/rfc-enforce-rule-of-three/89874/17) and [adding a non-blocking pre-commit clang-tidy bot](https://discourse.llvm.org/t/rfc-add-non-blocking-precommit-clang-tidy-analysis/89917/30).

* 许传奇： ChuanqiXu suggested that [C++20 modules serialisation tests be required for new features](https://discourse.llvm.org/t/rfc-clang-requiring-c-20-modules-serialization-tests-for-new-features/90768).

* Arm 最近更新量很大： A new pass was added to the AArch64 backend for code layout optimisation, forcing 64-byte function alignment in certain cases and instruction alignment for some instruction pairs that may cross cache lines.
  [8996639](https://github.com/llvm/llvm-project/commit/899663966c7e).

* 好久没有为win32开发过程序了： A bug was fixed where Clang over and under-aligned variables for Windows MSVC. [8ecec45](https://github.com/llvm/llvm-project/commit/8ecec455183f).

### 本周推荐阅读

本周推荐：《龙与洋鬼子：一位瑞典地质学家眼中的万象中国》

作者：安特生

译者：李雪涛

推荐：小编1号

最近微信读书推荐的民国的书籍变多了些，这本书的作者精力也是颇具“穿越”色彩。作为一个瑞典考古学家，来到了民国时期的中国，进行考虑，还受雇于北洋政府，跟袁世凯见过面。北京猿人、仰韶文化等发现都有作者的参与或主导。

所以阅读本书就时不时有一种年代缝合感，九年义务教育中已经忘记差不多的历史、地址、政治、语文知识，通过一个具体的老外考古学家串在了一起。
