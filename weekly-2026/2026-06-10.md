### LLVM/Clang/LLDB/LLD

本节内容来自 [LLVM Weekly 第 649 期](http://llvmweekly.org/issue/649)，
[Alex Bradbury](https://www.linkedin.com/in/alex-bradbury/) 大哥持续稳定输出。

* 宋教授笔耕不缀： Fangrui Song blogged about [recent improvements to LLVM's hash table implementations](https://maskray.me/blog/2026-06-07-recent-llvm-hash-table-improvements).

* 删除比添加要难： Luke Lau started an RFC thread on [removing trivial vector predication intrinsics](https://discourse.llvm.org/t/rfc-remove-trivial-vp-intrinsics/90972).

* 额，这个现在很容易吵起来了： Richard Pasek kicked off a discussion on [accepting vibecoded backend targets](https://discourse.llvm.org/t/rfc-add-vibecoded-ezh-target/90983). A majority of respondents are not keen, expressing concern about the maintenance burden.

* 日积跬步： 64-bit frame offsets can now be used for 64-bi RISC-V.
  [3f67f16](https://github.com/llvm/llvm-project/commit/3f67f166d658).

* 透明度+1: LLVM project governance documentation was added.
  [3c9afa9](https://github.com/llvm/llvm-project/commit/3c9afa9fe7c0).

* 进迭时空K3能用么： MC layer support was added for the proposed RISC-V integrated matrix tile load/store extensions.
  [0f18088](https://github.com/llvm/llvm-project/commit/0f18088dca8a).

### 本周推荐阅读

本周推荐：《视觉之旅：改变世界的机器（彩色典藏版）》

作者：西奥多·格雷

译者：徐大军、程翰璋

推荐：小编1号

如果你喜欢机械和技术，对内燃气等有兴趣，这本书非常适合在忙完了一阵之后坐在沙发里自习研究。人类工程上的很多巧思和灵机一动都在这本书里能够找到。烧开水的智慧从最早的磨盘到蒸汽机到核电站，都存在用视觉上可以理解的方式呈现在读者的眼前。前人的工程智慧真是值得被记录下来。
