### LLVM/Clang/LLDB/LLD

本节内容来自 [LLVM Weekly 第 622 期](http://llvmweekly.org/issue/622)，
[Alex Bradbury](https://www.linkedin.com/in/alex-bradbury/) 大哥持续稳定输出。

* 非常有价值： Matt Godbolt is doing a series of blog posts "[Advent of Compiler Optimisations"](https://xania.org/202511/advent-of-compiler-optimisation). The first one has [now been posted](https://xania.org/202512/01-xor-eax-eax).

* 宋教授！ Fangrui Song shared some [results from porting the Mach-O compact unwind format to ELF](https://discourse.llvm.org/t/rfc-improving-compact-x86-64-compact-unwind-descriptors/47471/45).

* 日积跬步： Rematerialization for scalar loads was enabled in the RISC-V backend.
  [4b35ff5](https://github.com/llvm/llvm-project/commit/4b35ff583fbc).

* 贡献社区的重要形式之一就是评审代码： Guidance for reviewing commit access requests was documented.
  [d7dcc10](https://github.com/llvm/llvm-project/commit/d7dcc108fc3a).

* Tenstorrent本周在上海发布了RISC-V CPU： A vector scheduling model was added for the Tenstorrent Ascalon D8 RISC-V processor.
  [ad3d9fb](https://github.com/llvm/llvm-project/commit/ad3d9fb3cab7).

* 日积跬步：Dwarf fission (`-gsplit-dwarf`) can now be used for RISC-V.
  [5f777b2](https://github.com/llvm/llvm-project/commit/5f777b2c8fac),
  [cc1c417](https://github.com/llvm/llvm-project/commit/cc1c41724dda).

* The BOLT inliner was extended to work on functions with pointer authentication.
  [bab1c29](https://github.com/llvm/llvm-project/commit/bab1c2971a31).
